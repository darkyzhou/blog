## thehub

- CPU: Intel Core i7-12700T (8P+4E cores, 20 threads)
- RAM: 2x16GB KLEVV BOLT X DDR4-3200
- GPU: SAPPHIRE Radeon RX 460 2G GDDR5 OC
- SSD: KIOXIA EXCERIA G2 1TB
- Motherboard: MAXSUN MS-CHALLENGER H610ITX 2LAN
- OS: Proxmox VE 8.2.7

> 这是我的第五代 All in One 主机，前四代的 CPU 分别是：
> - 第一代：Intel Core i3-9100
> - 第二代：Intel Core i5-8400T
> - 第三代：AMD Ryzen 5 5800H
> - 第四代：AMD Ryzen 3 3900X

这台主机运行了几台虚拟机，分别用于 x86 平台的日常开发、服务运行和黑苹果。

- 日常开发：Arch Linux，"I use arch btw"
- 跑服务：NixOS，直通了核显用于运行 Jellyfin，我的几乎全部服务都在上面
- 黑苹果：macOS Sonoma 14.16.1，直通了 RX 460，作为下载机

> 为什么要用黑苹果做下载机？因为 Windows 下的迅雷、百度网盘等软件太臃肿且丑了。
> MacOS 下的迅雷和 Windows 下的界面差距极大，前者更简洁，且几乎没有广告。
